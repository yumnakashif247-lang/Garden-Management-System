# Garden Management System

A C++ Garden Management System for managing plants, workshops, procurement records, and garden operations using AVL Trees, Queues, Singly Linked Lists, and the 0/1 Knapsack Algorithm.

## Features
- Plant Management
- Workshop Management
- Procurement Tracking
- Resource Optimization
- Data Structure-Based Operations

## Concepts Used
- AVL Tree
- Queue
- Singly Linked List
- 0/1 Knapsack Algorithm
